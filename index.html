This code creates three questions with multiple-choice options. When the user clicks the "Submit" button for each question, it checks if an option is selected and updates the score accordingly. The score is stored in the local storage, and an alert is shown with the current score. The CSS styling includes a centered heading and a light gray background for the page. Each question is styled with a white background, a border, and some padding. Feel free to customize the questions, options, and styling to fit your needs.
